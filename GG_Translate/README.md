# API Google Dịch cho ChatFuel
### Tham số
* *src_lang*: Ngôn ngữ của văn bản cần dịch
* *tar_lang*: Ngôn ngữ của văn bản kết quả
* *q*: Văn bản cần dịch
* Phương thức: GET
* Dữ liệu trả về: JSON
![API Google Dịch cho ChatFuel](https://i.imgur.com/oEdFg4l.png)
